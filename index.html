¿Que hace la web?

  • Permite ver cuales son las peliculas en tendencia y recomendadas segun la API de themoviedb
  • Puede buscar una pelicula en particular utilizando el buscador
  • Facilita filtrar peliculas por la categorias que tengan
  • Muestra detalles de las peliculas individuales como descripcion, duracion y peliculas relacionadas

¿Como lo hace?

  • Las tecnologias web utilizadas son HTML, CSS y Javascript
  • Tambien incluye la libreria de AXIOS para el manejo de API e implementa el framework de Bootstrap para el diseño

Caracteristicas

  • El diseño se adapta a todos los dispositivos, desde computadoras hasta celulares
  • Es una single-page application (SPA), o aplicación de página única, que significa que esta todo en una sola página con el propósito de dar una experiencia más fluida a los usuarios
  • Los recursos necesarios se cargan dinámicamente cuando lo requiera la página, normalmente como respuesta a las acciones del usuario
  • La terminacion de la URL de la web va cambiando de acuerdo a la seccion donde se encuentre
